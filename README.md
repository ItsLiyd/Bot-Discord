# 🛠️ Cara Install dan Jalankan Code Gwa

Buat lu yang mau nyobain kodingan ini, ikutin langkah di bawah.

### 1. Install Dulu Bahannya
Ketik di **bash** atau **apalah namanya itu** perintah di bawah ini buat install paket-paket yang diperluin (Discord, Database, dll):

```bash
npm install discord.js mongoose dotenv
```
### 2. Daftarin Command
Kalo slash command ga ke detect, coba bash ini sekali ajah (kalo lebih kadang commands nya dobel bjer 🗿)
```bash
node deploy-commands.js
```
### 3. Coba Jalanin Bot Nya
```bash
node index.js
```
